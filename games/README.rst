    * Initial Matrix
    ::

        - - -

        - - -

* Save Princess

    * Display path to princess

    * How to play
    ::

        $ cd games
        $ python save_princes.py

    * INPUT
        - N (int odd size of the matrix)
        - next N lines matrix array

    * Example Input
    ::

        m - mario
        p - princess

    ::

        $ python save_princes.py
        3
        --m
        ---
        p--

    * OUTPUT for above input
    ::

        DOWN
        DOWN
        LEFT
        LEFT
